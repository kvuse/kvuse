Money #

金额处理 #

默认小数点后两位

金额处理：88.8888
vue
<template>
  <span>
    <span>金额处理：</span>
    <span v-money>{{ value }}</span>
  </span>
</template>

<script setup>
import { ref } from 'vue';

const value = ref(88.8888);
</script>

小数点位数 #

设置point可以设置小数点位数 例如：v-model:point="4"

金额处理：88.88888
vue
<template>
  <span>
    <span>金额处理：</span>
    <span v-money.point="4">{{ value }}</span>
  </span>
</template>

<script setup>
import { ref } from 'vue';

const value = ref(88.88888);
</script>

数据更新 #

WARNING

如果有更新数据，谨慎使用，数据更新不会自动更新数据。

如果使用可以使用v-money:value="value"

金额处理：88.88888
vue
<template>
  <span>
    <span>金额处理：</span>
    <span v-money:value="value">{{ value }}</span>
  </span>
</template>

<script setup>
import { ref } from 'vue';

const value = ref(88.88888);

setTimeout(() => {
  value.value = 188.66666;
}, 1000);
</script>
