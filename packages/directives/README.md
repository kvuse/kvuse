# @kvuse/core

## 一些常用的 `vue3` 自定义指令

- vParams 一些常用的参数处理
- vButton 按钮点击事件
- vFocus 聚焦

## 安装

```bash
# Npm  
npm install @kvuse/directives
 
# Yarn  
yarn add @kvuse/directives

# Pnpm  
Pnpm install @kvuse/directives

```

## 使用

```js
const { vParams, vFocus, vButton, .... } from '@kvuse/directives'
```
