Button #

TIP

点击之后延时 800ms, 防止重复点击

基础使用 #

vue
<template>
  <kv-button>button</kv-button>
  <kv-button type="primary">
    primary-vant
  </kv-button>
  <kv-button type="success">
    success
  </kv-button>
  <kv-button type="danger">
    danger
  </kv-button>
  <kv-button type="warning">
    warning
  </kv-button>
  <kv-button type="info">
    info
  </kv-button>
</template>

调整尺寸 #

可设置size类型，控制按钮大小。可选值：large, default, small

vue
<template>
  <div>
    <div class="mb10">
      <kv-button size="large" type="primary">
        primary
      </kv-button>
    </div>
    <kv-button type="primary">
      normal
    </kv-button>
    <kv-button size="small" type="primary">
      small
    </kv-button>
    <kv-button size="mini" type="primary">
      mini
    </kv-button>
  </div>
</template>

<script setup></script>

<style lang="scss" scoped></style>

链接模式 #

可设置link类型，控制按钮链接模式

vue
<template>
  <kv-button link type="primary">
    primary
  </kv-button>
  <kv-button link type="success">
    success
  </kv-button>
  <kv-button link type="danger">
    danger
  </kv-button>
  <kv-button link type="warning">
    warning
  </kv-button>
  <kv-button link type="info">
    info
  </kv-button>
</template>
