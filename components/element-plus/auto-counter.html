AutoCounter 数字动效 #

基础使用 #

TIP

默认从0开始

0
vue
<template>
  <div class="width-240">
    <k-auto-counter v-model="count" class="font-20 font-bold" />
    <div class="mt20">
      <k-button type="primary" @click="setNum">
        设置随机数
      </k-button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const count = ref(20);

const setNum = () => {
  count.value = Math.ceil(Math.random() * 100);
};
</script>

设置开始值 #

如果设置start，可设置开始数值

20

设置start从10开始

vue
<template>
  <div class="width-240">
    <k-auto-counter v-model="count" :start="20" class="font-20 font-bold" />
    <p class="color-99">
      设置start从10开始
    </p>
    <div class="mt20">
      <k-button type="primary" @click="setNum">
        设置随机数
      </k-button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const count = ref(50);

const setNum = () => {
  count.value = Math.ceil(Math.random() * 100);
  console.log('count.value: ', count.value);
};
</script>

如果设置end，可设置结束值

是否开启动效 #

如果设置autoinit，可设置是否开启功效，默认true

关闭动效：
100
vue
<template>
  <div class="width-240">
    <div class="flex">
      <span>关闭动效：</span>
      <k-auto-counter v-model="count" :autoinit="false" class="font-20 font-bold" />
    </div>
    <div class="mt20">
      <k-button type="primary" @click="setNum">
        设置随机数
      </k-button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const count = ref(100);

const setNum = () => {
  count.value = Math.ceil(Math.random() * 100);
};
</script>

显示前后缀 #

如果设置prefix，显示前缀，设置suffix,显示后缀

前缀：￥0
后缀：0元
vue
<template>
  <div class="width-240">
    <k-auto-counter v-model="count" prefix="前缀：￥" class="font-20 font-bold" />
    <k-auto-counter v-model="count" prefix="后缀：" suffix="元" class="font-20 font-bold mt20" />
    <div class="mt20">
      <k-button type="primary" @click="setNum">
        设置随机数
      </k-button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const count = ref(20);

const setNum = () => {
  count.value = Math.ceil(Math.random() * 100);
};
</script>

小数点位数 #

如果设置decimals，可显示小数点位数

小数点两位：0.00元
vue
<template>
  <div class="width-240">
    <k-auto-counter v-model="count" prefix="小数点两位：" suffix="元" :decimals="2" class="font-20 font-bold mt20" />
    <div class="mt20">
      <k-button type="primary" @click="setNum">
        设置随机数
      </k-button>
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const count = ref(20);

const setNum = () => {
  count.value = Math.random() * 100;
};
</script>
