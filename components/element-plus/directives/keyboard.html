keyboard 监听键盘 #

TIP

设置v-keyboard,可监听键盘事件，进行事件处理

基础使用 #

监听所有键值：
当前键值：
vue
<template>
  <div v-keyboard:[eventListen].any>
    <span>监听所有键值：</span>
    <div>当前键值：<span class="c-red font-18">{{ currentKey.toLocaleUpperCase() }}</span></div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const currentKey = ref('');
const eventListen = (event) => {
  console.log('event: ', event);
  currentKey.value = event.key;
};
</script>

</style>

单键监听 #

监听Enter键值：
当前状态：按回车触发
vue
<template>
  <div v-keyboard:[eventListen]="{buttonKey:'Enter'}">
    <span>监听Enter键值：</span>
    <div>当前状态：<span class="c-red font-18">{{ currentState }}</span></div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const currentState = ref('按回车触发');
const eventListen = () => {
  currentState.value = '回车已触发';
  setTimeout(() => {
    currentState.value = '按回车触发';
  }, 800);
};
</script>

<style lang="scss" scoped>

</style>

多键监听 #

监听Enter、Shift键值：
当前状态：按键盘触发
vue
<template>
  <div v-keyboard:[eventListen]="{buttonKey:'Enter'}" v-keyboard:[shiftListen]="{buttonKey:'Shift'}">
    <span>监听Enter、Shift键值：</span>
    <div>当前状态：<span class="c-red font-18">{{ currentState }}</span></div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const currentState = ref('按键盘触发');

const eventListen = (event) => {
  currentState.value = `${event.key}已触发`;
  setTimeout(() => {
    currentState.value = '按键盘触发';
  }, 800);
};

const shiftListen = (event) => eventListen(event);
</script>

</style>

焦点下可监听 #

WARNING

设置v-keyboard,单键监听在输入框焦点下或者有elementUI弹框提示下不执行，如果执行，可设置修饰符focus,dialog参数

设置focus修饰符,可焦点下监听键盘事件

提示：一般情况下：封装的输入框都有`enter`事件，在这只是举个例子
输入框焦点下，监听Enter键值：
当前状态：按回车触发
vue
<template>
  <div v-keyboard:[eventListen].focus="{buttonKey:'Enter'}">
    <div class="c-red mb20">提示：一般情况下：封装的输入框都有`enter`事件，在这只是举个例子</div>
    <span>输入框焦点下，监听Enter键值：</span>
    <div class="width-240 mt10 mb10">
      <k-input placeholder="自动获取焦点" type="text" v-focus v-model="value"></k-input>
    </div>
    <div>当前状态：<span class="c-red font-18">{{ currentState }}</span></div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const value = ref('');

const currentState = ref('按回车触发');
const eventListen = () => {
  currentState.value = '回车已触发';
  setTimeout(() => {
    currentState.value = '按回车触发';
  }, 800);
};
</script>

</style>

弹框下可监听 #

设置dialog修饰符,可弹框下监听键盘事件

弹框下，监听F2键值：
按F2打开弹框
vue
<template>
  <div v-keyboard:[eventListen].focus.dialog="{buttonKey:'F2'}">
    <span>弹框下，监听F2键值：</span><br />
    <div class="mt10">
      <span class="mr20">按F2打开弹框 </span>
      <k-button type="primary" @click="visible = true">打开弹框</k-button>
    </div>
    <k-dialog v-model="visible" width="400px"></k-dialog>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const visible = ref(false);
const eventListen = () => {
  visible.value = !visible.value;
};
</script>
