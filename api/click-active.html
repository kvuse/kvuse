OnClickActive ​

点击元素选中，可设置选中的状态 ​

基础使用 ​

默认选中第一个

  • 微信
  • 支付宝
  • 其他
vue
<template>
  <ul class="flex">
    <li
      class="border-radius class-item mr20" v-for="item in list" :key="item.value"
      v-on-click-active="{ activeName:'active-item' }"
    >
      {{ item.label }}
    </li>
  </ul>
</template>

<script setup>
import { vOnClickActive } from '@kvuse/core';

const list = [
  { label: '微信', value: 1 },
  { label: '支付宝', value: 2 },
  { label: '其他', value: 3 },
];
</script>

<style lang="scss" scoped>
.class-item {
  border: 1px solid #eaeefb;
  padding: 10px 20px;
}

.active-item {
  border: 1px solid #f00;
  color: #f00;
}
</style>

设置默认 ​

可设置defaultIndex值设置默认选中，必须设置className

  • 微信
  • 支付宝
  • 其他
vue
<template>
  <ul class="flex">
    <li
      class="border-radius default-item mr20" v-for="item in list" :key="item.value"
      v-on-click-active="{ activeName:'active-default', className:'default-item', defaultIndex: 2 }"
    >
      {{ item.label }}
    </li>
  </ul>
</template>

<script setup>
import { vOnClickActive } from '@kvuse/core';

const list = [
  { label: '微信', value: 1 },
  { label: '支付宝', value: 2 },
  { label: '其他', value: 3 },
];
</script>

<style lang="scss" scoped>
.default-item {
  border: 1px solid #eaeefb;
  padding: 10px 20px;
}

.active-default {
  border: 1px solid #f00;
  color: #f00;
}
</style>
